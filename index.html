Projeto HTML: Tema - Client x Server e Tipos de Servidores

1. Client x Server

Nesta seção irei explicar a diferença entre Cliente e servidor, usando um exemplo comum da web:

  • Cliente: faz a requisição (ex. navegador).
  • Servidor: responde a requisição (roteia páginas, processa dados).

Nota: o protocolo mais comum é o HTTP.

Exemplo de requisição

O cliente solicita "http://meusite.com" e o servidor envia o HTML.

Definições

HTTP é o protocolo usado para transferir páginas web.

O HTTP é o "idioma" que navegador e servidor usam para se comunicar de forma padronizada.

2. Tipos de Servidor

Existem vários tipos, entre eles:

  1. Servidor Web (Apache, Nginx)
  2. Servidor de Banco de Dados
  3. Servidor de Aplicação (Node.js, Tomcat)

Se algum tipo se tornar obsoleto, pode ser descontinuado e substituído por alternativas mais modernas.

Exemplo: Apache 2.4

Versão Atual: Apache HTTP Server 2.4 48.

3. Hardware e Software em Servidores

3.1 Hardware

Componentes básicos:

  • Processador (CPU potente)
  • Memória RAM de alta capacidade
  • Armazenamento rápido (SSD, NVMe)
3.2 Software

Software comum em servidores:

  • Apache, Nginx (Web)
  • MySQL, PostgreSQL(banco de dados)
  • Docker, VirtualBox(containerização e virtualização)

Algumas tecnologias populares

Linux é o sistema operacional mais usado em servidores.
